# GUZU - Android Source Code

## OVERVIEW
## BY SHUBHAM MALLICK

GUZU is an Android application project developed as part of my learning and practical development journey in software engineering and Android app development.

This repository contains the complete source code of the application. The project was built to improve my understanding of Android development, UI structure, app logic implementation, debugging, and project management using GitHub.

The main objective behind this project was not only to build an app, but also to gain real-world development experience through experimentation, problem-solving, and continuous improvement.
